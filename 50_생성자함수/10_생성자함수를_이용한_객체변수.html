사용자정의 생성자함수

		const hong = {name:'홍길동', age:20};
		const duly = {name:'둘리', age:7};
		const hani = {name:'하니', age:15};
		
		위와같이 데이터구조가 같은 객체를 동적으로 만들어야 할때
		생성자함수 사용을 고려해 볼 수 있다.
		
		=> 생성자함수는 {key:value}의 쌍으로 된
		   객체변수를 생성하는 또다른 방법이다.
		   
		=> 프로퍼티와 메소드를 미리 설계도에 디자인 후,
		   new 키워드를 사용해서 객체변수를 생성한다.
		   
		=> 생성자함수의 이름은 대문자로 시작하는 관례
