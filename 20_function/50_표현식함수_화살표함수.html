표현식함수, 화살표함수

		1. 기명함수(선언식함수)
		
			function sumFn(a,b){
				console.log(a+b);
			}
		
			우리가 위와같이 선언식 함수로 작성하면,
			자바스크립트 해석기는 아래와 같은 '표현식함수'코드로 변환한다.
			
		2. 표현식함수는 변수에 익명함수를 할당하는 방식이다.
			const sumFn = fuction(a,b){
				console.log(a+b);
			}
			
		3. 화살표함수 - 기존의 함수선언 방식을 간소화 함
			const sumFn = (a,b)=>{
				console.log(a+b);
			}
